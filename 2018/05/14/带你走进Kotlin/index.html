kotlin简介

Kotlin 是一个用于现代多平台应用的静态编程语言 ，由 JetBrains 开发
Kotlin 1.0 正式版发布于2016年初，2017谷歌I/O大会：宣布 Kotlin 成 Android 开发一级语言，目前最新版本1.2.x

为什么选择 Kotlin？

  • 简洁：大大减少样板代码的数量、函数式（日常开发中至少可节省大约30%的时间）
  • 安全: 避免空指针异常（在生产环境出现异常的概率也降低了不少）
  • 互操作性: 充分利用JVM、Android
    和浏览器的现有库Kotlin可以编译成Java字节码，也可以编译成JavaScrip）
  • 工具友好: 可用任何 Java IDE 或者使用命令行构建
  • 协程：让你彻底告别回调(1.1+版本特性 还没用过- -。。感觉直接用rxjava也可以)

语法简洁

  • 支持类型推断
  • var表示变量，val表示常量简洁
  • ( ：) 类表示继承，方法后表示返回类型
  • 句尾不需要(;)

空指针安全

  • 在类型上的处理，即在类型后面加上?，即表示这个变量或参数以及返回值可以为null，否则不允许为变量参数赋值为null或者返回null
  • 对于一个可能是null的变量或者参数，在调用对象方法或者属性之前，需要加上?，否则编译无法通过。
